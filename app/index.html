  • Test js

    Trier des données et en faire la moyenne (json + js)

    Le fichier data.json représente les ventes d'un bar entre février 2016 et janvier 2018.

    Depuis ce fichier :

    1. Additionner les totaux en volumes vendus dans la catégorie "bière" au cours de l'année 2017
    2. Injecter par l'intermédiaire de javascript le résultat obtenu dans la balise 
      <span id="total"></span>
      du fichier test-js.html
  • Test html

    Mise en forme d'un formulaire de connexion simple (html + css)

    Il s'agit d'intégration pure, le fonctionnel n'est pas demandé

    Le formulaire doit :

    1. Respecter les maquettes graphiques fournies (dossier app/psd à la racine du projet)
    2. Être centré horizontalement et verticalement
    3. Être compatible navigateur recents (Firefox, Chrome, Safari, Internet Explorer... dernières versions)
    4. Être responsive

    Aucune intéraction n'est proposée dans la maquette, réalisation libre des effets hover, focus...
